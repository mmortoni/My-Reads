## Projeto My Reads

### Pré-requisitos

Para executar este aplicativo, você precisará do seguinte:

* Node.js
* Linha de comando (Windows: cmd / Mac: terminal)
* YARN ou NPM

Para executar:

```
yarn install
yarn start
```
ou
```
npm install
npm start
```

##### Abrir  http://localhost:3000/
